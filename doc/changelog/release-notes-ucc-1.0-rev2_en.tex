\section{UCC Images}

New official UCC images will be provided with UCC 1.0-rev2. These images
include all software updates up to UCC 1.0-rev2 and can be installed by
installing the packages \ucsName{ucc-1.0-rev2-thinclient-image}
respectively \ucsName{ucc-1.0-rev2-desktop-image}. Installing or
updating the UCC App automatically installs the new thinclient image
\ucsName{ucc-1.0-rev2-thinclient-image}. See the 
\href{http://docs.univention.de/ucc-manual-1.0.html#installation:imagemanagement}{UCC Manual}
for detailed information about UCC image management.

\section{UCC Integration Packages}
This section lists all changes to ucc integration packages, i.e. packages that are installed on Univention Corporate Server.
\subsection{ucc-image-toolkit}
\begin{itemize}
\item Removed network-manager from ucc-thinclient.cfg, a dependency has been added to \ucsName{univention-corporate-client} (\ucsBug{30297}).
\item Added default values for \ucsUCRV{ucc/mount} for desktop and thinclient image (\ucsBug{30552}).
\item \ucsName{ucc-image-set-join-information} now restricts file permissions (\ucsBug{30819}).
\item Renamed CD grub boot menu entries (\ucsBug{30464}).
\item Added additional boot option: Installation with forced repartition (\ucsBug{30464}).
\item Include img.md5 file on .iso if .img file is included (\ucsBug{30464}).
\item Create ldap.secret link file in image files (\ucsBug{31024}).
\item Dist-upgrade packages during image creation (\ucsBug{31028}).
\item Remove \ucsUCRV{kdm/autostart} setting from ucc-desktop.cfg for consistent plymouth-stop handling (\ucsBug{31127}).
\item New version for default images in image configs (\ucsBug{31147}).
%\item (\ucsBug{}).
\end{itemize}

\subsection{ucc-management-integration}
\begin{itemize}
\item Restart \ucsName{univention-management-console-server} in ucc-management-integration.postinst(\ucsBug{30439}).
\item Restart \ucsName{univention-s4-connector} in ucc-management-integration.postinst (\ucsBug{30845}).
\item Added settings/ucc\_session udm objects for UCC, UCC-remote and LXDE sessions (\ucsBug{30420}).
\item Remove UCC directory integration when deinstalling the UCC App (\ucsBug{31046}).
\end{itemize}

\subsection{ucc-pxe-boot}
\begin{itemize}
\item Added restart of rsyslog and univention-firewall during installation to allow ucc clients to log to server via syslog(\ucsBug{30456}).
\item Divert syslog entries from ucc clients to seperate files in /var/log/univention/ucc-clients/ for each host(\ucsBug{30040}).
\end{itemize}

\subsection{ucc-default-images}
\begin{itemize}
\item New default image packages \ucsName{ucc-1.0-rev2-thinclient-image} and \ucsName{ucc-1.0-rev2-desktop-image} for new default images (\ucsBug{31147}).
\end{itemize}

\section{Univention Corporate Client packages}
This section lists all changes to ucc packages, i.e. packages that are installed in an ucc image.
\subsection{univention-corporate-client}
\begin{itemize}
\item Added dependency to network-manager (\ucsBug{30297}).
\item Added symlink support to \ucsName{univention-ucc-sync-persistent-files} (\ucsBug{30463}).
\item Added DEBIAN\_FRONTEND to \ucsName{univention-ucc-software-update} (\ucsBug{31010}).
\item Exit softwareupdate script when a reboot is necessary (\ucsBug{31061}).
\item Added additional messages during software update (\ucsBug{31061}).
\item Unset \ucsUCRV{kdm/autostart} to keep plymouth displaying during software updates (\ucsBug{31061}).
\item remove unused kernel packages before apt-get (\ucsBug{31012}).
\end{itemize}

\subsection{univention-lightdm}
\begin{itemize}
\item Improved \ucsUCRV{}descriptions (\ucsBug{30933}).
\item Improved \ucsName{force-user-session} to support LXDE (\ucsBug{30420}).
\end{itemize}

\subsection{univention-ucc-join}
\begin{itemize}
\item Add \ucsName{/etc/ldap.secret} to persistent files (\ucsBug{30463}).
\item Recreate openssh-server hostkeys during join and add them to persistent files (\ucsBug{30163}).
\end{itemize}

\subsection{univention-ucc-remote-mount}
\begin{itemize}
\item Removed initial setting 'true' from \ucsUCRV{ucc/mount} (\ucsBug{30552}).
\item Ignore ram* and loop* devices in \ucsName{add\_fstab\_entry }(\ucsBug{30468}).
\end{itemize}

\subsection{univention-ucc-theme}
\begin{itemize}
\item Added dependency to libglib2.0-bin (\ucsBug{30579}).
\end{itemize}

\subsection{univention-ucc-initramfs}
\begin{itemize}
\item Start the loopback interface on system boot which improves boot times when the master ldap server is unreachable (\ucsBug{30411}).
\end{itemize}

\subsection{initramfs-tools}
\begin{itemize}
\item Do not backup old initramfs files (\ucsBug{31015}).
\item Create new initramfs files in temporary directory (not in /boot) (\ucsBug{31015}).
\item Temporary directory is /tmp or /ucc\_root (the dir with more free space, default /tmp) (\ucsBug{31015}).
\end{itemize}

\subsection{cups}
\begin{itemize}
\item Reconnect to cups server after connection timeout on the server (\ucsBug{30911}).
\end{itemize}
