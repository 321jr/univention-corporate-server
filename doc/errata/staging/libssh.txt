A new update is available for Univention Corporate Server 3.1 as
part of the extended security maintenance.
It addresses the following problem:

Program component:  libssh
Reference:          CVE-2014-0017 CVE-2016-0739
Fixed version:      0.4.5-3.15.201606090935

The following issues were fixed:
* The PRNG is not always correctly reseeding when a new process is forked
  (CVE-2014-0017).
* Weak Diffie-Hellman secret generation in libssh (CVE-2016-0739).
