Allgemein

UCS-Appliance ermöglicht es, Images von virtuellen Instanzen zu kopieren/klonen und die Grundkonfiguration des Systems beim Bootvorgang per Browser anzupassen. Die Grundkonfiguration beinhaltet alle notwendigen Einstellungen, damit das System der Domäne beitreten kann und somit direkt einsatzfähig ist. Darüber hinaus können weitere Einstellungen wie z.B. die Software-Ausstattung vorgenommen werden.

Konfigurationsschritte

Die Grundkonfiguration des Systems ist aufgeteilt in die folgenden Schritte:

  • Die Spracheinstellungen beinhalten alle sprachrelevanten Einstellungsmöglichkeiten wie Zeitzone, Tastaturlayout und Systemsprachen.
  • Basis-Einstellungen geben grundlegende Einstellungen wie Rechner- und Domänenname, LDAP-Basis, Windows-Domänename, und Root-Passwort vor.
  • Über die Netzwerk-Einstellungen können IP-Adressen (IPv4 und IPv6) sowie DNS-Server, Gateways und HTTP-Proxy festgelegt werden.
  • Nach Vorgabe der Zertifkatseinstellungen wird ein neues Root-Zertifikat für die Domäne erstellt. (Dieser Schritt ist nur auf DC-Master-Systemen notwendig.)
  • Über die Software-Einstellungen können bestimmte Software-Komponenten installiert und deinstalliert werden.

Währende des Speichervorgangs wird auch der Domänenbeitritt durchgeführt. Dazu ist auf allen System-Rollen außer DC-Master auch die Eingabe eines gültigen Domänen-Administrator-Accounts notwendig. Nach dem erfolreichen Domänenbeitritt kann der Einstellungs-Assistent geschlossen und der Boot-Vorgang fortgesetzt werden.

Werden die Einstellungen von einem externen System aus in einem Webbrowser vorgenommen, wird im Anschluss an die Konfiguratin um eine Neuanmeldung für UMC gefragt werden.
