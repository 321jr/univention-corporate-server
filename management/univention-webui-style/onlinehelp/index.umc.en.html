Univention Management Console Help

Univention Management Console is a modularly designed, web-based application for the administration of individual Univention Corporate Server systems. The application allows the user to control services, check system utilisation, and check or adjust system parameters. The web-based interface of UDM is described in detail in current manual (PDF) of Univention Corporate Server. Besides the manual the homepage of Univention provides additional documentation for Univention Corporate Server. These documents assist in different topics like installation or administration.

Answers to frequent questions, handy hints and solutions for UDM can also be found at Univention support database (SDB) or at Univention forum.

A detailed online help for Univention Directory Manager is in the pipeline and will be provided here upon completion.
